# Política de Privacidade - SEI-MCP Bridge

**Última atualização:** 25 de janeiro de 2025

## 1. Introdução

A extensão **SEI-MCP Bridge** ("Extensão") é desenvolvida e mantida por Iudex ("nós", "nosso"). Esta política descreve como coletamos, usamos e protegemos suas informações.

## 2. Informações Coletadas

### 2.1 Dados de Autenticação
- Credenciais de acesso ao SEI (armazenadas localmente no navegador)
- Tokens de sessão temporários

### 2.2 Dados de Uso
- Órgãos/unidades SEI acessados
- Funcionalidades utilizadas
- Erros e logs de diagnóstico

### 2.3 Dados de Pagamento
- Processados exclusivamente pelo Stripe
- Não armazenamos dados de cartão de crédito

## 3. Como Usamos os Dados

- **Autenticação:** Para manter sua sessão no SEI
- **Funcionalidade:** Para executar automações via MCP
- **Melhoria:** Para aprimorar a extensão
- **Suporte:** Para resolver problemas técnicos

## 4. Armazenamento de Dados

- Credenciais são armazenadas **localmente** usando `chrome.storage.local`
- Comunicação via WebSocket local (localhost)
- Não enviamos credenciais para servidores externos

## 5. Compartilhamento de Dados

**Não vendemos nem compartilhamos** seus dados pessoais, exceto:
- Com o Stripe para processamento de pagamentos
- Quando exigido por lei

## 6. Segurança

Implementamos medidas de segurança incluindo:
- Comunicação local via WebSocket (não sai da máquina)
- Armazenamento local seguro
- Sem transmissão de credenciais pela internet

## 7. Seus Direitos

Você pode:
- Acessar seus dados
- Solicitar exclusão de dados
- Revogar consentimento
- Exportar seus dados

## 8. Retenção de Dados

- Dados de sessão: Até logout
- Logs de erro: 30 dias
- Dados de assinatura: Enquanto a conta estiver ativa

## 9. Cookies e Rastreamento

A extensão **não utiliza** cookies de rastreamento ou analytics de terceiros.

## 10. Alterações na Política

Notificaremos sobre alterações significativas através:
- Atualização da extensão
- Email (se cadastrado)
